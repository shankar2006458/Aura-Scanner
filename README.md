# ✨ AuraScore

Discover your hidden energy with a fun and interactive aura scanner web app.

AuraScore is a stylish frontend project where users enter personal details, upload a photo, and receive a generated aura result with scores, personality vibes, lucky day, hidden strength, and more.

## 🌐 Live Demo
`https://shankaar2006458.github.io/AuraScore/`

---

## 🚀 Features

### 🎯 Core Experience
- 3 Free Credits for new users
- Personal Aura Scan based on user inputs
- Fun generated aura results
- Energy score out of 100
- Confidence level %
- Social magnetism %
- Lucky day + lucky number
- Hidden strength reveal
- Share result to clipboard

### 📸 Input System
- Upload photo (required)
- Name input
- Date of birth
- Gender
- Relationship status
- Work style
- Favorite color
- Mood today
- Favorite number

### 💎 Premium Upgrade Flow
- Credit system using LocalStorage
- Upgrade modal after free credits end
- Multiple plan selection
- QR code payment popup
- UPI payment support
- Promo code support
- Manual credit unlock system

### 🎨 UI / UX
- Premium glassmorphism design
- Animated particle background
- Smooth page transitions
- Responsive mobile-first layout
- Stylish popups
- Aura ring animations
- Toast notifications
- Modern gradients & glow effects

## 💡 How It Works

1. User gets 3 free credits  
2. Fill details + upload photo  
3. Click **Reveal My Aura**  
4. Loading animation starts  
5. Aura result appears  
6. Credits reduce after each scan  
7. When credits end, upgrade popup appears
